Instructions About Creator

This is Wordament. Click to insert a local text file of your choice. Wait
fpr the timer to start. Click and drag across the board
to select words. Each letter that you select in either direction
will accumulate for points, which will be shown up in the scoreboard.
There is a timer that counts down the time, so make sure you look quickly
and try to find as many words as possible! When the timer runs out, refresh
the page for a new game!
